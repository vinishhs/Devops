Docker is a platform that allows applications to run inside containers.

A container packages:

Application code

Dependencies

Runtime

Configuration

This guarantees the application runs the same in development, testing, and production.

Why Docker?

Portable across systems

Lightweight compared to VMs

Faster startup

Consistent environments

Easy scaling

Important Docker Terms
Term	Meaning
Image	Template used to create containers
Container	Running instance of an image
Dockerfile	Script containing build instructions
Registry	Storage for images (e.g., Docker Hub)
